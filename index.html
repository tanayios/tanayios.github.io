Homedics Drift (Wi-Fi)

Project Overview

Designed a wellness-focused iOS app for Homedics Drift smart sand tables, allowing remote control and personalization via Wi-Fi connectivity.

Key Features

  • Remote control of sand patterns and lighting via Wi-Fi
  • RGBIC lighting with thousands of color combinations
  • Built-in focus & nature soundscapes
  • Curated and user-defined pattern playlists

Technical Details

  • Wi-Fi & BLE support using Network and CoreBluetooth

Achievements

  • Enabled seamless user experience with real-time control
  • Contributed to successful product launch of ST-250/200 models
  • Delivered scalable, modular code for future expansion
